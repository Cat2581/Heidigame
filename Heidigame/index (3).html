This game was made for my level 2 project at The League of Amazing Programmers.

1. Why I Made This Game

I decided to make this game because the format of other memory games has always peeked my interest so I wanted to make one of my own!

2. What I Learned

Some things they gave feedback on was color choices they were very bright and take away from the game so I brought the brightness down a little to make the game more enjoyable.

3. Architecture of My Game

Well its a pretty simple memory game but the way I customized it was by the prestation and the way the game is played it is a simple.

4. Technical Implementation

I'm actually really proud of making a custom cursor I think it give the game more personality and I really like that.

5. Most Challenging Part

The most challenging part of it all was trying to get the transition of the fruits to go smoothly it was a bit confusing to get to work.

6. Future Fixes and Updates
